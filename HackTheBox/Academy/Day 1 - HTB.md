# HACK THE BOX (DAY 1) - Intro To Academy

## Outcome
Introduction to Hack The Box Academy and how the platform/modules work.

## Task
Use the provided Pwnbox terminal to identify the Linux OS flavour using:

cat /etc/issue

## Mistake Made
I initially typed:

cat/etc/issue

This caused an error because Linux commands require a space between the command and the file path.

## Correct Command
cat /etc/issue

## Lesson Learned
Linux commands are very syntax-sensitive, and even small formatting mistakes can cause errors.
