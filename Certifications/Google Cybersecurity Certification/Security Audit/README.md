# Internal Security Audit & Compliance Assessment

## Project Overview
Conducted a comprehensive internal security controls audit for Botium Toys to identify critical risk exposures, analyze administrative and technical vulnerabilities, and provide high-leverage mitigation strategies.

## Core Objectives & Framework Alignment
*   **Business Continuity & Recovery:** Evaluated organizational disaster plan gaps using the modern **NIST Cybersecurity Framework (NIST CSF 2.0)**, mapping controls directly to Data Security (`PR.DS`) and Recovery Planning (`RC.RP`).
*   **Regulatory Compliance Analysis:** Performed framework gap assessments to measure corporate alignment with **PCI DSS** (cardholder data privacy) and **GDPR** (EU customer data classification and 72-hour breach notification protocols).
*   **Technical Mitigation Design:** Engineered strategic solutions to eliminate single points of failure, including proposing the installation of an **Intrusion Detection System (IDS)**, deploying a centralized password management infrastructure, and establishing an isolated, encrypted **3-2-1 backup pipeline** to safeguard critical SPII and financial data.

## Portfolio Distinction Note
This project repository deliberately enhances the official course assignment guidelines by upgrading all defensive recommendations to align directly with modern **NIST CSF 2.0 core functions**, ensuring the audit metrics reflect real-world enterprise standards.
