Linux Shells (Tryhackme module)

Notes:

echo $SHELL = See which shell you are currently using
 

shells

Bash - bourne again shell
Fish - friendly interactive shell
zsh - Z shell

.sh = default extension for bash scripts

nano = create file command

#! = shebang (should be at the start of every script) #! followed by the name of the interpreter to use while executing the script)

e.g. #!/bin/bash

chmod +x = give script execution permission
./ = tells the shell to execute the script

e.g. "./first_script.sh"

loop code:

do = indicates start of loop code
done = end of loop code
