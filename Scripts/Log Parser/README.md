# Log Parser Project - Initial Preparation

## Project Aim

The aim of this project is to create a beginner Python log parser to further practise Python programming while developing foundational defensive security scripting skills.

This project will help build familiarity with:
- Python scripting
- Log analysis
- Defensive cybersecurity concepts
- File parsing and data handling

The long-term goal is to gradually progress toward more advanced defensive security projects and improve overall programming confidence.

---

## Planned Features

Current ideas for the project include:
- Reading log files
- Detecting failed login attempts
- Counting suspicious events
- Identifying repeated IP addresses
- Printing alerts for suspicious activity

---

## Why I Am Doing This Project

I wanted to begin creating practical cybersecurity-related projects while improving my Python knowledge through hands-on learning.

This project is intended to serve as both:
- a beginner scripting project
- and an introduction to defensive security automation concepts.
