# Log Parser

## Project Overview

This project is a beginner Python log parser created to develop foundational Python programming skills while learning defensive security scripting concepts.

The parser reads a log file line by line and identifies `ERROR` and `WARNING` entries, helping demonstrate how security analysts can filter important events from system logs.

---

## Features

- Reads log files using Python
- Processes logs line by line
- Detects `ERROR` and `WARNING` events
- Displays matching log entries in the terminal

---

## Skills Demonstrated

- Python file handling
- Loops and iteration
- Conditional statements
- Basic log analysis
- Defensive security scripting
